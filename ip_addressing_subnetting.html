IP Addressing & Subnetting

IP Addressing is the system of assigning unique identifiers (IP addresses) to devices on a network. Subnetting divides large networks into smaller, more manageable segments to improve efficiency and security.

üåê IP Addressing

An IP address consists of two parts: network ID and host ID.

Example (IPv4): 192.168.1.10 (where 192.168.1 is the network and 10 is the host)

IPv4 vs IPv6:

  • IPv4: 32-bit (e.g., 192.168.1.1)
  • IPv6: 128-bit (e.g., 2001:0db8:85a3::8a2e:0370:7334)

üìä Subnetting

Subnetting helps divide a network into multiple smaller sub-networks (subnets) using subnet masks.

Subnet Mask Example: 255.255.255.0

Benefits of Subnetting:

  • Efficient IP address utilization
  • Improved network performance
  • Enhanced security and isolation

üõ† CIDR Notation:

Classless Inter-Domain Routing (CIDR) simplifies subnetting using slash notation:

Example: 192.168.1.0/24 means subnet mask 255.255.255.0

  • Enables structured network design
  • Optimizes routing and switching
