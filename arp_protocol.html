ARP Protocol

ARP (Address Resolution Protocol) plays a critical role in local networks. It allows devices to discover each other's MAC addresses given their IP addresses, enabling successful communication within the same network segment.

üß© Why is ARP Needed?

Even though devices communicate using IP addresses at the Network Layer, actual data transmission on local networks happens using MAC (hardware) addresses. ARP bridges this gap by mapping IP addresses to MAC addresses.

üõ† How ARP Works:

  1. Device A wants to send data to Device B (knows its IP but not MAC).
  2. Device A broadcasts an ARP request on the local network: "Who has IP 192.168.1.5?"
  3. Device B replies with its MAC address: "192.168.1.5 is at MAC AA:BB:CC:DD:EE:FF".
  4. Device A updates its ARP table and sends data directly using the MAC address.
