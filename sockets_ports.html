Sockets & Ports

Sockets and Ports are crucial elements that allow multiple network applications to run simultaneously on the same machine without interfering with each other.

üß© What is a Socket?

A socket is a combination of an IP address and a port number. It uniquely identifies a network connection endpoint.

Example: 192.168.1.5:80 (IP:Port)

üß™ What is a Port?

A port is simply a number assigned to specific processes or services on a device. It helps identify which application should handle the incoming or outgoing data.

üõë Common Port Numbers

  • 80: HTTP (Web traffic)
  • 443: HTTPS (Secure web traffic)
  • 21: FTP (File Transfer Protocol)
  • 25: SMTP (Email Sending)
  • 53: DNS (Domain Name System)
  • 22: SSH (Secure Shell)
